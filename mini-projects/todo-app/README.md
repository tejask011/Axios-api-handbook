# Todo App (Mini Project)

A straightforward, interactive Todo application built with Vanilla JavaScript and Axios.

## Objectives
- Fetch a list of todos from an external API.
- Add new todos using a POST request.
- Delete todos using a DELETE request.
- Manage loading states and error feedback.

## Technology Stack
- **Frontend**: HTML5 and CSS3.
- **Logic**: JavaScript (ES6+).
- **HTTP Client**: Axios (integrated via CDN).
- **Public API**: [JSONPlaceholder](https://jsonplaceholder.typicode.com)

## Implementation Instructions
To execute this project, open the `index.html` file in any modern web browser.

## Key Learning Points
1. Utilizing `axios.get()` with query parameters for data limiting.
2. Sending structured data using an `axios.post()` request.
3. Dynamically updating the Document Object Model (DOM) following a successful API response.
