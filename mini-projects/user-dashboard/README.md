# User Dashboard (Mini Project)

A dynamic dashboard implementation that retrieves and displays user profiles with integrated pagination.

## Objectives
- Fetch paginated datasets from a REST API.
- Implement functional "Previous" and "Next" navigation controls.
- Utilize array mapping to render component-based lists from API data.
- Develop a responsive grid-based layout.

## Technology Stack
- **Frontend**: HTML5 and CSS3 (utilizing CSS Grid and Flexbox).
- **Logic**: JavaScript (ES6+).
- **HTTP Client**: Axios.
- **Public API**: [ReqRes](https://reqres.in)

## Implementation Instructions
To execute this project, open the `index.html` file in any modern web browser.

## Key Learning Points
1. Managing API pagination through query parameters.
2. Implementing loading states during network operations.
3. Synchronizing the user interface with the application's current data state.
